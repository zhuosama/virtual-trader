## 一、当前报告问题分析

### 1. 盘前报告问题

**当前格式**：
```
📊 盘前分析报告
市场基调: bearish

⚠️ 风险审查: 需要修改
风险警告:
  ⚠️ main账户前3大持仓占比68.4%超过限制50.0%
  ⚠️ lab账户前3大持仓占比77.7%超过限制50.0%

🎯 今日目标: 修改交易计划后重新审查
```

**问题**：
- ❌ 信息堆砌，缺乏层次
- ❌ 市场分析信息缺失
- ❌ 交易计划不清晰
- ❌ 重要信息不够突出

### 2. 盘后报告问题

**当前格式**：
```
📊 盘后复盘报告
📉 main账户: -885元
📈 lab账户: +2,530元
今日交易: 1笔
发现错误: 1个

有效做法:
  ✅ 今日执行了1笔交易
  ✅ 日报中包含有价值的洞察

失败做法:
  ❌ main账户格力电器仓位11.1%超过限制10.0%

🔧 策略更新:
  max_single_position: 0.1 -> 0.08

📝 Changelog: 1条记录
```

**问题**：
- ❌ 缺乏汇总摘要
- ❌ 数据格式不直观
- ❌ 重要性分级不清晰
- ❌ 行动项不明确

## 三、优化原则

### 1. 结构原则

**层次清晰**：
```
标题 → 摘要 → 详情 → 行动项
```

**信息分组**：
- 相关信息分组显示
- 使用分隔线区分不同部分
- 重要信息优先显示

**视觉引导**：
- 使用emoji增强可读性
- 关键数据加粗或特殊标记
- 颜色语义：📈绿色(好) 📉红色(差) ⚠️黄色(警告)

### 2. 数据原则

**格式化**：
- 数字：1,234,567元（千位分隔）
- 百分比：+1.23% / -0.45%（正负号）
- 比率：37.2% / 62.8%（清晰对比）

**对比显示**：
- 今日 vs 昨日
- 实际 vs 目标
- 当前 vs 限制

**重点突出**：
- 关键指标单独显示
- 异常数据特别标记
- 变化幅度清晰标注

### 3. 行动原则

**明确性**：
- 每个行动项有明确的执行者
- 每个行动项有明确的执行时间
- 每个行动项有明确的执行标准

**优先级**：
- 紧急重要：🔴
- 重要不紧急：🟡
- 紧急不重要：🟢
- 不紧急不重要：⚪

**可执行性**：
- 行动项具体可操作
- 包含必要的上下文信息
- 提供执行所需的资源
