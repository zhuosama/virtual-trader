# 风控规则

## 主账户（保守型）

| 规则 | 限制 |
|------|------|
| 单笔最大仓位 | 总资产 10% |
| 单板块最大仓位 | 总资产 30% |
| 总持仓上限 | 总资产 80% |
| 个股止损线 | 成本价 -7% |
| 账户最大回撤 | -10% 触发全面减仓至 50% 以下 |
| 单日最大交易次数 | 5 笔 |

## 实验账户（宽松型）

| 规则 | 限制 |
|------|------|
| 单笔最大仓位 | 总资产 15% |
| 单板块最大仓位 | 总资产 40% |
| 总持仓上限 | 总资产 90% |
| 个股止损线 | 成本价 -10% |
| 账户最大回撤 | -15% 触发全面减仓至 50% 以下 |
| 单日最大交易次数 | 8 笔 |

## 通用规则

- 任何交易前检查风控，不符合则拒绝交易并记录原因
- 止损为硬性规则：触发止损线的持仓在下一个交易日必须卖出
- 回撤减仓后需要等待账户从回撤低点回升 3% 才能恢复正常仓位水平
- 风控参数本身可在周末复盘时调整，但调整需记录到 changelog 并说明理由
